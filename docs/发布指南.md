## 📤 发布到 GitHub Release（详细步骤）

### 步骤 1: 准备 DMG 文件

你已经有了 `Eason-v1.0.0.dmg`，可以直接使用！

如果需要重新创建：

```bash
# 第一步：用最新代码重新编译 .app
./scripts/build_app.sh

# 第二步：打包成 DMG（版本号自己定）
./scripts/create_dmg.sh 2.4.7
```
或者一键搞定：

```bash
./scripts/release.sh 2.4.7
```

如果需要 ZIP 格式（可选）：
```bash
cd /Users/code/Eason/dist
zip -r Eason-v1.0-macOS.zip Eason.app
```

### 步骤 2: 创建 Release

1. 访问你的 GitHub 仓库：
   ```
   https://github.com/你的用户名/Eason
   ```

2. 点击右侧的 **Releases**

3. 点击 **Create a new release**

4. 填写信息：

   **Tag version**: `v1.0.0`  
   **Release title**: `🎉 Eason v1.0 - 首个正式版本`  
   **Description**:
   ```markdown
   ## 📧 Eason v1.0

   首个正式版本发布！

   ### ✨ 功能特点

   - 📬 联系人自动采集（从 163 邮箱）
   - 📨 批量邮件发送
   - ⚙️ 智能批次控制（防止触发反垃圾邮件机制）
   - 📊 实时进度显示
   - 💾 配置自动保存
   - 🎨 精美的 macOS 原生界面

   ### 📥 下载

   选择下面的 **Eason-v1.0.0.dmg** 下载（推荐）

   或者下载 ZIP 格式：`Eason-v1.0-macOS.zip`

   ### 💻 系统要求

   - macOS 10.14 或更高版本
   - 163 邮箱账号及授权码

   **DMG 安装（推荐）**：
   1. 双击 `Eason-v1.0.0.dmg` 打开

   1. 下载并解压 `Eason-v1.0-macOS.zip`
   2. 双击 `Eason.app` 启动
   
   首次运行可能需要在「系统偏好设置」→「安全性与隐私」中允许

   ### 📖 文档

   - [使用手册](https://github.com/你的用户名/Eason#readme)
   ```

## ⚠️ 首次运行注意事项

用户首次运行时可能遇到以下情况：

### 情况 1: "无法打开，因为无法验证开发者"

**解决方法**：
1. 右键点击 Eason.app
2. 选择"打开"
3. 在弹出的对话框中点击"打开"

### 情况 2: "已损坏，无法打开"

**解决方法**：
```bash
# 在终端执行（告诉用户）
xattr -cr /Applications/Eason.app
```

### 解决方案: 代码签名（可选，需要 Apple Developer 账号）

```bash
# 如果你有开发者证书
codesign --force --deep --sign "你的证书名称" dist/Eason.app

# 验证签名
codesign --verify --verbose dist/Eason.app
```

## 💡 提示

### 减小应用体积

当前 71 MB 已经很不错了！如果想进一步优化：

```bash
# 使用 UPX 压缩（可选）
brew install upx
pyinstaller --upx-dir=/usr/local/bin ...
```

### 创建 DMG 安装包（高级）

```bash
# 安装 create-dmg
brew install create-dmg

# 创建 DMG
create-dmg \
  --volname "Eason" \
  --window-pos 200 120 \
  --window-size 800 400 \
  --icon-size 100 \
  --app-drop-link 600 185 \
  Eason-v1.0.0.dmg \
  dist/Eason.app
```
