# ⚡️ 5分钟快速开始

## 🎯 目标

在 5 分钟内完成第一次邮件发送！

---

## 📝 准备清单

在开始之前，请确保：

- [ ] 有一个 163 邮箱账号
- [ ] 已安装 Python 3.8+
- [ ] 已下载本项目

---

## 🚀 三步开始

### Step 1: 一键安装（1分钟）

打开终端，执行：

```bash
cd /Users/code/Eason
./install.sh
```

脚本会自动安装所有依赖（虚拟环境 + PyQt6）。

---

### Step 2: 获取授权码（2分钟）

1. 浏览器打开：https://mail.163.com
2. 登录邮箱
3. 点击「设置」→「POP3/SMTP/IMAP」
4. 开启「IMAP/SMTP服务」
5. 记下生成的**授权码**（例如：`ABC123XYZ456`）

⚠️ 这不是你的登录密码！

---

### Step 3: 启动程序（30秒）

```bash
./run.sh
```

或者：

```bash
python3 email_assistant_gui.py
```

程序界面打开后：

1. 填入邮箱：`your_email@163.com`
2. 填入授权码：`ABC123XYZ456`
3. 点击「📥 采集联系人」标签
4. 点击「🚀 开始采集」

完成！ 🎉

---

## 🎓 进阶操作（可选）

### 采集联系人

采集完成后：
1. 点击「💾 导出联系人」
2. 保存为 `contacts.txt`

### 发送测试邮件

1. 切换到「📤 批量发送」标签
2. 点击「📂 浏览...」加载 `contacts.txt`
3. 填写：
   - 主题：`测试邮件`
   - 正文：`这是一封测试邮件`
4. 设置批次：`3 封/批`，间隔 `15 秒`
5. 点击「🚀 开始发送」

## ⚠️ 常见错误

### 错误 1: "No module named 'PyQt6'"

**解决**：
```bash
./install.sh
```

或手动：
```bash
source .venv/bin/activate
pip install PyQt6
```

### 错误 2: "Permission denied: ./run.sh"

**解决**：
```bash
chmod +x run.sh
./run.sh
```

### 错误 3: "Authentication failed"

**原因**：授权码错误或未开启 IMAP 服务

**解决**：
1. 重新获取授权码
2. 确保开启了 IMAP/SMTP 服务
