# 📧 邮件批量发送助手

## 🎉 欢迎使用！

你的邮件发送程序已经完成封装！现在拥有一个精美的 macOS 图形界面应用。

---

## ⚡️ 快速开始（3步）

### 1️⃣ 一键安装

```bash
cd /Users/code/Eason
./install.sh
```

### 2️⃣ 启动程序

```bash
./run.sh
```

或手动激活虚拟环境：

```bash
source .venv/bin/activate
python email_assistant_gui.py
```

### 3️⃣ 获取163邮箱授权码

1. 访问 https://mail.163.com
2. 设置 → POP3/SMTP/IMAP
3. 开启服务，获取授权码
4. 在软件中填入授权码

完成！ 🎊

## 💡 使用技巧

### ✅ 推荐配置

| 发送量 | 批次大小 | 间隔时间 |
|--------|---------|---------|
| < 50封 | 10封/批 | 20秒 |
| 50-200封 | 10封/批 | 30秒 |
| > 200封 | 5封/批 | 60秒 |

### ⚠️ 注意事项

- 授权码不是登录密码！
- 首次使用建议小批量测试
- 间隔时间不要太短
- 邮件内容避免敏感词

## 🐛 遇到问题？

### 常见错误

**"No module named 'PyQt6'"**
```bash
./install.sh
```

**"Authentication failed"**
- 检查授权码是否正确
- 确认已开启 IMAP/SMTP 服务

**发送失败**
- 检查网络连接
- 增加批次间隔时间
- 检查邮箱地址格式

更多问题查看 [USER_GUIDE.md](USER_GUIDE.md#常见问题)
